# Student Pass/Fail Prediction using Machine Learning

##  Description
This project is a Machine Learning application that predicts whether a student will pass or fail based on the number of study hours. It uses Logistic Regression, a classification algorithm, to determine the result.

##  Objective
To build a model that can classify students as Pass or Fail based on study hours.

##  Technologies Used
- Python 3.x
- Pandas
- Scikit-learn

##  Project Structure
- main.py → Main Python file
- data.csv → Dataset
- requirements.txt → Libraries required
- README.md → Documentation

## Environment Setup
Install dependencies using:
pip install -r requirements.txt

##  How to Run
Run the following command:
python main.py

## Input and Output
- Input: Study hours
- Output: Pass or Fail

Example:
Enter study hours: 3  
Output: Fail

##  Concepts Used
- Machine Learning
- Logistic Regression
- Classification
